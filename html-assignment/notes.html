Notes for Reference

Quote
This allows you to add a small quotation within other content i.e. the quick brown fox jumps over the lazy dog.
Horizontal Line
This allows you to add a horizontal line to thematically break up content and should not be used for decorative lines.
Image
This allows you to insert an image into your HTML and requires both the src and alt tags.
Form
This allows you to create forms that allow for client server communication such as a search bar or contact information field.
Nav
This allows you to create clickable links that will allow access to other pages on the website.
